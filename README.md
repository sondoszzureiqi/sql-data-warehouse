# Medallion Data Warehouse
A comprehensive **Data Warehousing and Analytics** solution built on **SQL Server**, following the **Medallion Architecture** (Bronze, Silver, Gold layers). This project demonstrates building an end-to-end ETL pipeline, data modeling, and generating actionable insights.

## Features

- **Medallion Architecture**  
  - Bronze: Raw data ingestion  
  - Silver: Cleaned and transformed data  
  - Gold: Analytics-ready data for reporting  
![Architecture](docs/Architecture.jpg)

- **ETL Pipelines**  
  - Extract, Transform, Load processes using SQL Server  

- **Analytics & Insights**  
  - Aggregated and summarized data for dashboards and reporting
